Introduction

Pumped Functions

A modern TypeScript functional programming library that provides container-based dependency injection and reactive programming patterns. Built with type safety and developer experience in mind.

Why Pumped Functions?

  • 🔧 Container-based DI: Elegant dependency injection without the complexity
  • ⚡ Reactive Programming: Built-in reactivity with automatic dependency tracking
  • 🎯 Type Safety: Full TypeScript support with excellent inference
  • 🔄 React Integration: Seamless React hooks and components
  • 📦 Modular Design: Use only what you need
  • 🧪 Testing-First: Comprehensive testing utilities included

Key Features

Executors

Executors are the core building blocks - containers that hold values and define how dependencies are resolved.

import { provide } from '@pumped-fn/core-next';
 
// Simple executor
const greeting = provide(() => 'Hello World!');
 
// Executor with dependencies
const userGreeting = provide((name: string) => `Hello ${name}!`, name);

Reactive Programming

Built-in reactivity that automatically tracks dependencies and updates when they change.

const counter = provide(() => ({ count: 0 }));
const doubled = provide((state) => state.count * 2, counter.reactive);

React Integration

Seamless integration with React components using hooks and providers.

import { useResolves, ScopeProvider } from '@pumped-fn/react';
 
function App() {
  const [greeting, count] = useResolves(greeting, counter);
  return <div>{greeting} - Count: {count.count}</div>;
}
