Example 2: Todo App with Filters

🎯 Challenge

Can you add a “Clear Completed” button that removes all completed todos?

💡 Show Hint

Add this to the todoController:

clearCompleted: () => ctl.update(todos => 
  todos.filter(todo => !todo.completed)
)
