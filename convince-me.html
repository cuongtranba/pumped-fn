“How Does Testing Really Work?”

Let’s compare testing the same feature:

Redux Testing 😰

// 30+ lines of mocking and setup
const mockStore = configureStore(middlewares);
const store = mockStore({ counter: 0 });
 
store.dispatch(increment());
const actions = store.getActions();
expect(actions[0]).toEqual({ type: 'INCREMENT' });
 
// Mock selectors, providers, etc...

Pumped Functions Testing 😊

// 5 lines. That's it.
const scope = createScope(preset(counter, 5));
const controller = await scope.resolve(counterController);
await controller.increment();
expect(await scope.resolve(counter)).toBe(6);

No mocking. No providers. Just data.

// RTK: Still concepts to learn

  reducers: {
    increment: state => { state.value += 1 }
  }

// Pumped: Just functions
